# 更新日志

### 优化
- 优化了资源目录结构，采用更扁平的组织方式
- 统一了图标和图片的存放位置

- 删除了冗余的 `/assets/ui` 目录

### 修复
- 修复了图片引用路径问题
- 优化了错误处理机制
- 移除了调试代码和冗余日志
- 统一了错误提示方式

### 技术优化
- 使用 showError 替代 console.error
- 保留必要的错误处理机制
- 优化了代码结构和可维护性
